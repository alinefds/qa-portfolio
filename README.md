# A Projeto de automação web utilizando Robot Framework e Selenium.

Objetivo:
Automatizar o fluxo de cadastro de usuário na aplicação Automation Exercise.

Tecnologias:
- Robot Framework
- SeleniumLibrary
- FakerLibrary
- Python

Funcionalidades:
- Cadastro de usuário
- Geração dinâmica de massa
- Validação de dados
- Exclusão de conta
- Tratamento de anúncios/iframe

Execução:
robot -d results tests
